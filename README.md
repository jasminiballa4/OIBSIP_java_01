# 🚂 Online Reservation System

A browser-based Online Reservation System built with pure HTML, CSS, and JavaScript — no frameworks, no backend needed.

## Features

- **Login Form** — Secure login with ID and password validation
- **Dashboard** — Live stats (total bookings, confirmed, cancelled) + bookings table
- **Reservation System** — Book tickets with passenger details, train number (auto-fills train name), class type, date, and route
- **Cancellation Form** — Look up any booking by PNR number and cancel with one click

## How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. Login with: **ID = admin** | **Password = 1234**

## Modules

| Module | Description |
|--------|-------------|
| Login | Validates user credentials before granting access |
| Dashboard | Shows booking stats and recent reservations table |
| Book Ticket | Fill in details and generate a PNR instantly |
| Cancel Ticket | Search by PNR → view details → confirm cancellation |

## Tech Used

- CSS3 (custom dark theme)
- Vanilla JavaScript
